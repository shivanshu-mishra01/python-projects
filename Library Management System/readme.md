# 📚 Library Management System

A console-based **Library Management System** developed using **Python**, **Object-Oriented Programming (OOP)**, and **JSON File Handling**. This project allows users to manage library books by adding, searching, borrowing, returning, and removing books while maintaining persistent storage using a JSON file.

---

## 🚀 Features

* Add new books
* View all available books
* Search books by Book ID
* Borrow books
* Return borrowed books
* Remove books from the library
* View book transaction history
* Store data permanently using JSON
* Menu-driven console interface

## 💻 Functionalities

### 📖 Add Book

* Add a new book with a unique Book ID
* Store Book Name, Author Name, and Total Copies
* Prevent duplicate Book IDs

### 📚 View All Books

* Display all books available in the library

### 🔍 Search Book

* Search books using Book ID
* Display complete book details

### 📕 Borrow Book

* Borrow a book if copies are available
* Automatically decrease available copies

### 📗 Return Book

* Return a borrowed book
* Automatically increase available copies

### ❌ Remove Book

* Delete a book from the library

### 📝 Book History

* Track activities such as:

  * Book Added
  * Borrowed
  * Returned
