# 🏦 Bank Management System

A console-based **Bank Management System** developed using **Python** and **Object-Oriented Programming (OOP)**. This project simulates basic banking operations and stores user data permanently using **JSON File Handling**.

## 📌 Project Overview

This application allows users to create multiple bank accounts, securely log in using a PIN, perform banking transactions, and save account information even after the program is closed.

The project was built to strengthen core Python programming concepts such as OOP, dictionaries, loops, functions, conditional statements, and file handling.

---

## ✨ Features

* Create multiple bank accounts
* Secure login using Name and PIN
* Deposit money
* Withdraw money
* Check current account balance
* Change account PIN
* View transaction history
* Logout functionality
* Menu-driven console interface
* Persistent data storage using JSON
